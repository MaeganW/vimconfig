# Vim Config

## Getting Started

Clone down repo and replace the ~/.vim directory

Install Vundle

```
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim

```

Open up vim and install plugins:

```
:PluginInstall
```
